# qiitago
Unofficial qiita api converter for go
* https://qiita.com/api/v2/docs#%E6%8A%95%E7%A8%BF
* support v2
* under development, some struct support only.

## Usage
* go get 
```
go get github.com/ynishi/qiitago
```
* import
```
import github.com/ynishi/qiitago
```
